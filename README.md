# AngularDock
Docker Angular production environment

Dependencies
----------------------
1. Angular-CLI
2. Docker Engine, Docker-Compose

Getting Start
----------------------
	ng new myproject
	cd project
	npm install
	ng build --prod
	git clone https://github.com/itoonx/angulardock.git
	cd ./angulardock
	docker-compose build
	docker-compose up -d
